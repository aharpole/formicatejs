formicate.js

Turn spans into inputs in your form element.

Requirements
jQuery

Usage
In main.js.coffee

    $('#your_form_object').formicate() if $('#your_form_object')

yourform.html

    <form id="your_form_object">
        <span class="waiting">I want everyone to have the ability to update this text.</span>
    </form>
